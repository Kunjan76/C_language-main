 3 Key to understand  the Function
    ... Return Value..........
    1. int always used to return the value of the function
    2. void doesn't return a value of the function.

    ........Arguments.........
    This is the variable passed to the function
    1.With Argument means the  main function passed the value to user define function.
    2. without argument means the function doesn't pass the value.

    .......Where to print ........
    If the function doesn't return a value then print on the user defined function
    If the  Userdefined function return a value then print on the main function 




 IN local language (nepali)
